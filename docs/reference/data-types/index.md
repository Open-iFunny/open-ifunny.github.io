## Users

All data types sent by the iFunny API regarding user accounts. This includes all user profile information and some hidden information like ban information

[More: user-types.md →](user-types.md)

## Client, OAuth & Basic Token

Authentication data types including account info, bans, and appeals. Learn how to generate and prime basic tokens, authenticate with OAuth, and manage account access.

## Content, Comments & Tasks

Everything related to creating, posting, and interacting with content on iFunny. Includes the upload task workflow, comment threading, and content state management.

## Search & Discovery

Data types for searching users and content, browsing trending tags, and discovering curated content through compilations and digests.
