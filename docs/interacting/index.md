# Interacting with iFunny

Learn how to work with the iFunny API through practical, step-by-step examples using the official libraries. Each guide builds on the previous one, showing you how to navigate real-world scenarios.

All examples use one of three approaches: **cURL** with jq for manual HTTP requests, **TypeScript** with [iFunny.ts](https://github.com/makeshiftartist/iFunny.ts), or **Go** with [ifunny-go](https://github.com/open-ifunny/ifunny-go).
